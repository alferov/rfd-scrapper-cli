# rfd-scrapper-cli
> Read the lastest posts on redflagdeals from your terminal

## Installation
```sh
$ npm install rfd-scrapper-cli --global
```

## Usage
```sh
  $ rfd-scrapper [--limit=<number>]
    --limit=<number> Limit number of returned entries (default: 10)
```
